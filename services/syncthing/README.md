# Syncthing

When adding folders:
- Ignore permissions
- Disables comparing and syncing file permissions. Useful on systems with nonexistent or custom permissions (e.g. FAT, exFAT, Synology, Android).

On the server:
- Disable relay.
- Disable Global Discovery
